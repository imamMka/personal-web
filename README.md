A modern, professional personal portfolio built with **Next.js**, **Tailwind CSS**, and **Framer Motion**. This website showcases my expertise as a **Fullstack Developer** and **UI/UX Designer**, focusing on end-to-end digital solutions.

## 🛠 Technology Stack
- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/)
- **Components:** Custom Neumorphism Design
- **Deployment:** Static Site Generation (SSG)


## 🛠 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone [repository-url]
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production (Static Export):**
   ```bash
   npm run build
   ```
